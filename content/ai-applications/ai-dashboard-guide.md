# 如何用 AI 制作数字化学术会议看板

## 一句话解释

利用“数据脱敏 + Claude Artifacts 交互渲染”的工作流，你可以在一分钟内，将一份干瘪的学术会议医生签到 Excel 表格，变成一个包含漏斗图、趋势图和点击交互筛选的“高大上”数字会议数据看板。

## 适合谁

* 负责每个季度对线上/线下学术会议效果（ROI、到会率、互动深度）进行复盘的市场部数字化团队。
* 需要向高层汇报学术活动转化漏斗（受邀 -> 报名 -> 到会 -> 留存 -> 处方观念转化）的品牌组经理。
* 拥有大量数据，但不会写复杂 BI 代码（如 PowerBI/SQL）的业务助理。

## 核心工具组合

1. **Claude (带有 Artifacts 功能)**：这是目前全球制作交互看板最神奇的工具。它能根据数据直接生成一个可运行、可点击的网页级 Dashboard 组件，你可以直接预览并下载为静态网页文件。
2. **Excel**：用于存放原始数据并进行简单的分类整理。

---

## 极速实战工作流（以“三季度医生线上学术研讨会看板”为例）

### 第一步：准备并脱敏你的 Excel 数据
安全第一！在上传任何数据给 AI 之前，必须清除所有医生的姓名、电话和医院具体名称，仅保留省份、科室和参会时长。
* 在 Excel 中保留以下列：`省份`、`医生科室`、`参会状态（受邀/报名/到会）`、`参会时长（分钟）`、`互动提问次数`。
* 另存为 CSV 格式或直接全选复制。

### 第二步：将数据发给 Claude 并激活 Artifacts 看板生成
* 打开 [Claude 官网](https://claude.ai/)（确保设置中的 Artifacts 开关已打开）。
* 发送 Prompt：
  ```text
  你是一位资深的医药数字化营销数据分析师。我这里有一份“第三季度医生线上学术研讨会”的脱敏参会行为数据。
  请根据以下数据，为我设计一个美观、具有现代感（深蓝色医药主题调性）的交互式数据看板（Dashboard）：
  数据如下：
  [粘贴你的Excel数据，或者直接上传CSV文件]

  看板功能要求：
  1. 包含 3 个核心 KPI 卡片：总受邀人数、实际到会率（%）、平均参会时长（分钟）。
  2. 一个转化漏斗图（受邀 -> 报名 -> 到会），直观展示流失环节。
  3. 一个科室分布柱状图，支持悬停点击查看具体科室到会详情。
  4. 采用现代 Glassmorphism（玻璃拟态）卡片布局，使用 Chart.js 或 Recharts 渲染。
  ```

### 第三步：在浏览器中预览与交互
* Claude 会在右侧的 **Artifacts 窗口** 中直接渲染出这个交互看板。
* 你可以把鼠标悬停在图表上查看数字，点击筛选按钮切换省份。
* 点击 Artifact 窗口下方的 **“Publish”（发布）** 或 **“Download”（下载）** 按钮，将看板下载为一个自包含的 `index.html` 文件。

### 第四步：本地双击直接使用汇报
* 以后每次开季度复盘会，你不需要打开庞大的 BI 软件，直接在电脑里双击这个 `index.html` 文件，就能向老板演示带有丰富动效和点击交互的专业数据看板！

## 推荐 Prompt

你可以直接复制这套 Prompt，发给 Claude 快速帮你用 Mock 数据生成一个用于演示的看板原型：

```text
请为我制作一个用于医药市场部汇报的“数字化医生运营看板”交互原型。
要求在 Artifact 窗口输出一个单 HTML 文件，包含：
1. 顶部：看板标题“医生全渠道数字化互动看板”及三个指标卡片（总注册医生、活跃率、处方观念转化ROI）。
2. 中部：一个折线图，展示过去12个月医生在学术网站的月度活跃趋势。
3. 下部：一个科室到会雷达图（展示心血管科、内分泌科、肾脏科等医生的到会深度对比）。
设计要求：界面干净，配色使用代表医学和健康的科技蓝绿渐变色。
```
