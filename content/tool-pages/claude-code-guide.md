# Claude Code 学习指南

## 一句话解释

Claude Code 是由 Anthropic 官方推出的命令行 AI 编程助手，直接运行在你的终端（命令行）中，能够扫描整个代码库、自主运行本地测试、主动调试报错并执行多文件跨文件的代码修改。

## 适合谁

* 已经熟悉命令行操作，希望 AI 能够拥有更高自主权（Agent 级别）的数字化团队成员。
* 需要对庞大项目进行跨文件 Bug 诊断、自动化测试并自动修复开发环境的开发人员。

## 首次使用流程

### 第一步：安装运行环境
确保你的电脑中已经安装了 Node.js（v18 或更高版本）。你可以通过在终端输入 `node -v` 来检查安装情况。

### 第二步：全局安装包
打开你的 Windows PowerShell 或终端命令行，运行以下全局安装命令：
```bash
npm install -g @anthropic-ai/claude-code
```

### 第三步：启动与浏览器授权
切换到你的项目根目录下，运行启动命令：
```bash
claude
```
首次启动时，终端会弹出一个网页授权链接，登录你的 Anthropic 账号并粘贴返回的授权码即可。

### 第四步：下达任务开始对话
直接在终端输入你的任务需求：
```text
“帮我排查为什么启动脚本在 Windows 上会报错，并修复它。”
```

---

## 常见错误

* **在没有 Git 的目录下运行**：Claude Code 修改代码是直接在本地文件上生效的。如果项目没有 Git 版本控制，一旦 AI 改错将无法一键撤销。
* **过度放权导致费用超支**：在执行自动化测试时，如果不给 AI 限制尝试步数，它可能会为了修一个 Bug 自主循环执行几十次命令，消耗大量 API Token 费用。

---

## 防泄密安全提醒

* **敏感配置文件泄露**：Claude Code 在扫描项目时，默认会读取项目根目录下的所有文本文件。**必须在 `.gitignore` 中锁死 `.env`、`secrets.json` 等含有真实密钥的文件**，防止它们被作为上下文上传到云端大模型。
