# Figma 规则驱动的学习地图视觉重建设计

## 目标

在不改变 Agent 学习地图的课程内容、阅读器、搜索、进度、Prompt 生成器、UI/UX 课程链路及现有数据的前提下，使用 `awesome-design-md/design-md/figma/DESIGN.md` 的规则重建整站视觉表达，并在完成后推送当前 `main` 分支。

## 规则源与转译原则

本次唯一规则源为 Figma 的 DESIGN.md。项目不使用 Figma 名称、Logo、原始文案、产品截图或专属素材；只吸收其公开的系统级规则，并将其转换为“AI 学习工作台”的专属语言。

保留当前绿色拼贴课程主题，但不再使用旧的玻璃拟态、背景渐变、半透明浮层或大面积柔光阴影。页面将改为“黑白编辑框架 + 单块粉彩学习章节”的节奏：白色用于工具与阅读，淡彩色用于故事性课程模块，黑色只承担主操作、导航和反白信息区。

## 设计令牌

- 画布与文字：`#ffffff` / `#000000`；正文层级通过字重而非灰色透明度区分。
- 分隔与弱表面：`#e6e6e6` / `#f7f7f5`。
- 章节色块：浅绿 `#dceeb1`、淡紫 `#c5b0f4`、奶油 `#f4ecd6`、薄荷 `#c8e6cd`、粉色 `#efd4d4`、珊瑚 `#f3c9b6`、深靛蓝 `#1f1d3d`。
- 强调色：仅在一个限定提示或完成标记中使用洋红 `#ff3d8b`；完成状态使用绿色 `#1ea64a`。
- 字体：继续复用本地 Instrument Sans 作为无专属字体的显示/正文替代，JetBrains Mono 仅用于全大写（或中文等效的紧凑）章节标签、编号与说明。
- 间距：统一为 4、8、12、16、24、32、48、96px；最大内容宽度提升至 1280px。
- 圆角：文本按钮一律胶囊形；输入框和小内容块为 8px；章节色块和大型卡片为 24px；不再使用旧页面普遍的 30px+ 圆角。

## 页面结构

1. **顶栏与首屏**：白色、细分隔线、黑色主按钮；首屏回归白色画布。保留当前课程拼贴、路线标签、搜索、进度和快捷入口，但将其重新组织为贴纸式平面构图，取消玻璃效果。
2. **章节节奏**：相邻模块之间以白色画布和 96px 呼吸间距分隔。每个大章节只使用一种粉彩色块，避免同屏多种颜色竞争。建议：Prompt 生成器用浅绿、AI 应用用奶油、开发路径用深靛蓝、工具库用薄荷、PRD 用珊瑚、UI/UX 实操用淡紫。
3. **卡片与表单**：学习卡片为白色或弱表面、细边框、低阴影；彩色章节不依赖阴影制造层次。表单使用 8px 圆角、白底、细边框与清晰黑色焦点环。阅读器改为白色文档面，左侧目录与正文以发丝线分隔。
4. **导航与反馈**：导航在桌面使用清晰文字链接及一个黑色主操作；窄屏折叠为可访问的菜单按钮。所有主/次操作遵循黑白胶囊按钮体系，完成状态保留绿色徽记，键盘焦点使用可见黑色或洋红轮廓。
5. **动效与视差**：保留用户已要求的三层滚动位移和贴纸轻微旋转，但只作用于装饰和课程预览，正文与操作保持稳定。减少动态效果模式关闭视差、旋转和非必要过渡。

## 响应式规则

- `1280px`：主内容宽度封顶，保持多列课程网格。
- `960px`：四列降为两列，顶栏切换至菜单按钮，阅读器从双栏改为单栏。
- `768px`：粉彩章节色块撑满可视宽度并移除外侧圆角，课程卡保持一列或两列的可读宽度。
- `560px`：首屏显示标题缩至约 48px，胶囊按钮全宽，所有触控目标至少 44px。

## 验收标准

1. 所有原有业务内容、流程、数据键和阅读器功能仍可使用。
2. 页面不再保留玻璃拟态、渐变背景或旧式柔光卡片风格。
3. 顶栏、按钮、卡片、表单、阅读器、导航、章节色块及状态反馈均使用统一令牌。
4. 每个彩色章节色块均使用一类指定颜色，色块之间有白色间隔。
5. UI/UX 三课卡、原始 HTML 演示、线框图和 Stitch brief 入口仍可访问。
6. 桌面、平板、手机不产生横向溢出；移动端导航、触控目标和阅读器可用。
7. `prefers-reduced-motion: reduce` 时没有视差与旋转动效。
8. 构建脚本和 UI/UX 结构校验通过，且新增视觉令牌和响应式标记可被静态检查。
